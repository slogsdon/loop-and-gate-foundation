# Profiles

Reference docs that teach the agent *how you write* and *how you judge*, built
by the `profile-interview` skill.

- `voice-profile.md` — your writing DNA. Read before drafting any prose.
- `taste-profile.md` — your design and product judgment. Read at gates where
  the agent decides what's on-brand, what reads as slop, what's worth building.

Neither exists until you run `profile-interview`. Re-run it anytime to deepen
or correct a profile — it merges, it doesn't wipe.
